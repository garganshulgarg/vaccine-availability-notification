# vaccine-availability-notification
This application is used to validate cowin.gov.in to identify the vaccine availability an notification is triggered via SMS to respective mobile number.


Clone this repo in your local.

you can execute the application by execution following command through command line

mvnw spring-boot:run


## Prerequisite

Java 8 should be installed in your machine.
JAVA_HOME should be set as expected.


AWS Credentials should be configured to use AWS SNS Service
